## FINAL PROJECT FOR STATISTICAL MACHINE LEARNING COURSE 

### 1. Goal
Goal of this project is to analyze given set of data. First step is to perform linear regression using [LASSO](https://en.wikipedia.org/wiki/Lasso_(statistics)) method for [SURE](https://en.wikipedia.org/wiki/Stein%27s_unbiased_risk_estimate) estimator.

### 2. Files
* CV_NN.csv and Data.csv - given data
* Linear_regression.ipynb - jupyter notebook for linear regression
* Data_for_neural_network - remainder of data with linear part subtracted
* Neural_network.ipynb - jupyter nootebook with neural network analyzing the remainder
* Project_presentation.pdf - presentation with full annd detail description of the projrct and the results
